# OpenCV & Halcon canny

### 🎯 目标

在轮廓检测时发现OpenCV受边缘伪影的影响会出现识别到的线条偏移

了解到Halcon能够处理亚像素级别的轮廓，遵循MVP原则对比两个库对轮廓识别的精度，探究Halcon能否解决边缘伪影带来的影响

### 🛠 技术要点

- C++   OpenCV    Halcon
- 支持随机噪声

### 🔥实现思路

- 分别使用

- 在设置选项中绑定各个关键参数，实现动态调整参数的效果，以确保能够针对图像的分辨率给出合理的参数。

### 🤖 开发方式

使用Trae进行开发，代码使用 AI 生成。

### 📝 结果

使用Halcon库能够捕捉到效果更好、更稳定的轮廓线，而使用OpenCV则会因为边缘伪影导致对摄像头摄取图形轮廓不稳定。
